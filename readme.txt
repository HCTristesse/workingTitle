WORKING TITLE README

Setup Tutorial:
https://bohr.wlu.ca/cp411/eclipseCDT.php

Copy freeglut/include and freeglut/lib folders to MinGW/freeglut.
In project settings under C/C++ Build/Settings, add freeglut/include to include paths.
For libraries, add opengl32, glut32 and glut; add freeglut/lib to lib search paths.
